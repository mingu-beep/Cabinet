## 1. 프로젝트 개요

> 사용자 간의 자유로운 중고 품목 거래를 도와주는 서비스
> 
> 사용자가 직접 물건을 등록할 수 있고, 거래 방식을 직거래 혹은 사물함(cabinet) 이용으로 선택할 수 있다.

- 국비 학원 졸업 팀 프로젝트
- 개발 기간 : 5주 (2021.08 ~ 2021.09)

## 2. 프로젝트 

1. 사용 기술 <br>
  Web : Java,  Spring boot, BootStrap, Thymeleaf, HTML, CSS, JavaScript, jQuery, Ajax, MyBatis <br>
  DB : MySQL <br>
  Arduino : Arduino Sketch <br>
  형상 관리 : Git, Github
2. 요구 사항 <br>

    <img width="70%" alt="web" src="https://github.com/mingu-beep/Cabinet/assets/82128032/2935249f-2c58-4e04-9b7c-7d52401de715"/>

3. 구현 기능 <br>
  - 거래 게시판 : 등록 시간 / 조회수 순으로 오름차순 및 내림차순 정렬, 판매 완료 필터링 기능, 이미지/가격을 포함한 게시글 작성 <br>
  - 검색 기능 : 카테고리 및 지역 지정 가능<br>
  - 고객 센터 : 1대1 문의 게시판, 전체 공지 기능 구현<br>
  - 채팅방<br>
  - 관리자 기능 : 이벤트 관리, 카테고리 관리, 서비스 지역 관리, 회원관리<br>
